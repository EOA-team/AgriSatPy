agrisatpy.io package
====================

The agrisatpy.io package is the core element for interacting with any kind of raster data. It uses rasterio
as the backend for reading and writing raster data in an object-oriented manner.

The agrisatpy.io package and its SatDataHandler class are used by all functions of AgriSatPy that handle raster
data. The SatDataHandler class can also be used outside of the AgriSatPy's operational functionalities and provides
an object-oriented access to raster data including a set of powerful methods for raster data manipulation, extraction
and conversion.
